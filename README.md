# Fullstack Todo App

app description

## Run app

Run client:

```bash
npm run client
```

Run backend:

```bash
nodemon -e ts --exec tsx server/server.ts
```

Run both client and backend:

```bash
npm run dev
```

## Testing

Run tests:

```bash
npm run test
```

## Demo User
